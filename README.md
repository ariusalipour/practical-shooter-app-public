Practical Shooter App puts the IPSC rulebooks, scoring calculators, match verification tools, and learning resources in your pocket — offline, always.

## Features

- **Rulebooks** — Full IPSC rulebooks with offline download, search, glossary tooltips, cross-references, and revision comparison.
- **Event Calendar** — Find IPSC matches and events with filters by discipline, level, and country. Favourite events for quick access.
- **Scoring Tools** — Hit Factor Calculator, Stage Calculator, and Match Verifier for scoring and verifying match results.
- **Learn** — Articles covering practical shooting topics, quizzes to test your knowledge, and an AI-powered Rules Assistant (experimental).
- **Account** — Sync favourites, preferences, and settings across devices.

## Getting Started

Download the app from the [App Store](https://apps.apple.com/app/practical-shooter-app/id6478183643) or [Google Play](https://play.google.com/store/apps/details?id=com.barneymcgrew.practicalshooterapp), or try the [web demo](https://practicalshooter.expo.app).

> The web version supports rulebook browsing and learning features. The calendar and certain advanced features require a native device.
